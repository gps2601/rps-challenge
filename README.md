# Rock Paper Scissors Lizard Spock

Instructions
-------
- Use the following to get Rock Paper Scissors Lizard Spock running:
```
git clone https://github.com/gps2601/rps-challenge.git
cd rps-challenge
bundle
rackup
```
 - Visit [localhost:9292] to play the game!
